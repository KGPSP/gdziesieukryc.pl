# Gdzie się ukryć  
 
„Gdzie się ukryć” to nowoczesna aplikacja webowa przygotowana przez Biuro Informatyki i Łączności Komendy Głównej Państwowej Straży Pożarnej. Jej celem jest szybkie wskazanie najbliższych punktów schronienia (PS) w sytuacjach zagrożenia – takich jak silny wiatr, atak chemiczny czy inny nadzwyczajny incydent.  
 
## Czym są Punkty Schronienia  
 
Punkty Schronienia to istniejące obiekty budowlane lub ich części zwiększające bezpieczeństwo ludzi w chwilach zagrożenia, m.in. przejścia podziemne, tunele, stacje metra i garaże podziemne.  
 
## Funkcje aplikacji  
 
- lokalizuje najbliższe Punkty Schronienia i wyświetla ich adresy;  
- pozwala wyznaczyć trasę dojścia na mapie;  
- działa również w trybie offline po wcześniejszym pobraniu danych;  
- umożliwia zapisanie punktów w ulubionych;  
- zastępuje wcześniejszą aplikację „Schrony” i korzysta z bazy ok. siedemdziesięciu tysięcy zweryfikowanych obiektów.
